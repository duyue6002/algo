# NOTE

## 位运算

### 位运算符

左移，<<

右移，>>

按位或，|

按位与，&

按位取反，~

按位异或，^

```js
x^0 = x
x^1s = ~x
x^(~x) = 1s(~0)
x^x = 0
```

```js
// 判断奇偶
x % 2 === 1	 => (x & 1) === 1
x % 2 === 0	 => (x & 1) === 0
// 除2
x >> 1 	=> x = x / 2
// 清零最低位的1
x = x & (x - 1)
// 得到最低位的1表示的数
x & -x
x & ~x === 0
```
